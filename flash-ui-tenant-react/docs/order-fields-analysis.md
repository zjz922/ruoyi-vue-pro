## 二、API返回字段（来自/order/orderDetail接口）

### 核心金额字段
| 字段名 | 类型 | 说明 |
|-------|------|------|
| actual_receive_amount | String | 商家实收金额（分） |
| actual_receive_amount_details | Array | 实收金额明细 |
| author_cost_amount | String | 达人承担金额 |
| shop_cost | String | 商家承担金额 |
| platform_cost | String | 平台承担金额 |
| sum_amount | String | 订单总金额 |
| post_amount | String | 运费金额 |
| coupon_amount | String | 优惠券金额 |
| shop_coupon_amount | String | 店铺优惠券金额 |
| platform_coupon_amount | String | 平台优惠券金额 |
| voucher_deduction_amount | String | 抵扣金额 |

## 三、财务统计所需核心字段映射

### 订单明细页面字段映射
| 页面字段 | CSV字段 | API字段 | 计算公式 |
|---------|--------|---------|---------|
| 订单号 | 主订单编号 | shop_order_id | 直接取值 |
| 子订单号 | 子订单编号 | order_id | 直接取值 |
| 客户姓名 | 收件人 | post_receiver | 直接取值（脱敏） |
| 当前状态 | 订单状态 | order_status_desc | 直接取值 |
| 付款金额 | 订单应付金额 | pay_amount | 直接取值 |
| 商品成本 | - | - | 需要成本配置表 |
| 服务费 | 手续费 | service_fee | 直接取值 |
| 达人佣金 | - | promotion_amount | 直接取值 |
| 运费险 | - | freight_insurance | 直接取值 |
| 运费 | 运费 | post_amount | 直接取值 |
| 退款 | - | refund_amount | 直接取值 |
| 设置金额 | - | settle_amount | 直接取值 |
| 预估收益 | - | - | 付款金额 - 商品成本 - 服务费 - 达人佣金 - 运费 |
| 下单时间 | 订单提交时间 | create_time | 直接取值 |
| 完结时间 | 订单完成时间 | finish_time | 直接取值 |
| 结算时间 | - | settle_time | 直接取值 |

### 订单统计页面汇总字段
| 统计指标 | 计算公式 |
|---------|---------|
| 订单数 | COUNT(订单) |
| 付款金额 | SUM(订单应付金额) |
| 商品成本 | SUM(商品成本) |
| 服务费 | SUM(手续费) |
| 达人佣金 | SUM(达人佣金) |
| 运费 | SUM(运费) |
| 退款 | SUM(退款金额) |
| 预估收益 | SUM(预估收益) |

## 五、数据勾稽关系

```
订单明细 ──────────────────────────────────────────────────────────────
    │
    ├── 按日期汇总 ──> 订单统计（日统计）
    │                    │
    │                    ├── 按月汇总 ──> 按月汇总统计
    │                    │
    │                    └── 按年汇总 ──> 按年汇总统计
    │
    ├── 按30天汇总 ──> 最近30天明细
    │
    └── 费用分解 ──> 费用中心
                       │
                       ├── 服务费 ──> 平台服务费统计
                       ├── 达人佣金 ──> 达人佣金统计
                       ├── 运费 ──> 物流费用统计
                       └── 退款 ──> 退款统计
```

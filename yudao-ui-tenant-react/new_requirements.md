# 抖店财务应用新需求整理 (基于 deepseek_html_20260110_9fc9a3.html)

## 1. 核心架构与流程
- **架构层级**：表现层、应用层、服务层、数据层、集成层
- **核心流程**：业务事件驱动 -> 财务核算中枢 -> 资金流并行处理 -> 税务风险监控 -> 管理决策支持 -> 统一可视化呈现

## 2. 功能模块详解

### 2.1 实时经营总览仪表盘 (Module 1)
- **核心KPI**：总营收、总毛利、净利润、回款金额、可用资金、库存总价值、综合税负率 (刷新频率≤10分钟)
- **多维度穿透**：时间、平台、店铺、品类、品牌、区域 (3级穿透)
- **趋势与预警**：对比昨日/同期/预算/行业，红/黄/绿三色预警
- **口径转换**：统一财务口径 (如京豆、抖币、佣金的映射)
- **数据监控**：完整性校验 (差异率≤0.1%)

### 2.2 精细化财务核算模块 (Module 2)
- **收入确认**：支持权责发生制 (发货) / 收付实现制 (到账) / 双轨对比
- **成本匹配**：动态成本匹配 (加权平均/FIFO/个别计价)，出库实时触发
- **费用归集**：自动归集平台费、推广费、物流费等，支持分摊
- **三大报表**：实时生成利润表、资产负债表、现金流量表

### 2.3 资金管理模块 (Module 3)
- **流水识别**：智能分类收入/支出/转账
- **回款管理**：跟踪回款周期，预警逾期
- **资金预测**：ARIMA/机器学习预测未来7/30/90天资金流
- **风险监控**：余额安全线、大额支付预警

### 2.4 库存与成本联动模块 (Module 4)
- **成本计价**：实时更新库存成本
- **出库匹配**：订单出库实时计算成本
- **周转分析**：库龄分析、积压预警 (>90天)、缺货预警

### 2.5 管理会计分析模块 (Module 5)
- **盈利分析**：多维度组合 (时间x店铺x品类等)
- **ROI计算**：推广费用投入产出比
- **预算管控**：进度预警、超支预警
- **本量利分析**：盈亏平衡点计算、场景模拟

### 2.6 费用自动归集模块 (Module 6)
- **多源抓取**：API/账单/手工
- **智能分摊**：按销售额/订单量/固定比例分摊
- **校验机制**：防遗漏、金额校验

### 2.7 综合税负率计算模块 (Module 7)
- **税金预提**：实时预提增值税、所得税等
- **风险管理**：税负率异常监控
- **税务报告**：生成多维度税务分析报告

## 3. 页面与交互调整计划

### 3.1 导航结构调整
- **新增/调整菜单**：
    - **经营概览** (对应 Module 1)
    - **财务核算** (对应 Module 2) -> 子菜单：收入确认、成本核算、财务报表
    - **资金管理** (对应 Module 3) -> 子菜单：资金流水、回款分析、资金预测
    - **库存成本** (对应 Module 4) -> 子菜单：库存计价、周转分析
    - **经营分析** (对应 Module 5) -> 子菜单：盈利分析、ROI分析、预算管理、本量利分析
    - **费用中心** (对应 Module 6) -> 子菜单：费用归集、分摊规则

## 数据导入结果
- 成功导入72条商品成本数据（来自滋栈官方旗舰店_商品成本.csv）

## 功能验证

### 1. 数据展示 ✓
- 商品号、商品信息、店铺、售价、成本价、毛利率、库存、生效日期均正常显示
- 分页功能正常（共72条记录，分4页显示）

### 2. 搜索功能 ✓
- 支持按商品号、商品名称、商家编码搜索

### 3. 筛选功能 ✓
- 支持按配置状态筛选（全部状态/已配置/未配置）
- 支持按店铺筛选

### 4. 编辑功能 ✓
- 点击编辑按钮可修改成本价
- 支持保存和取消操作

### 5. 新增功能 ✓
- 点击"新增商品"按钮可添加新的商品成本配置
- 支持填写商品号、商品标题、成本价、售价、商家编码、店铺等信息

### 6. 删除功能 ✓
- 点击删除按钮可删除商品成本配置（软删除）

### 7. 历史记录 ✓
- 点击历史按钮可查看成本变更历史

## 检查方法

### 1. 功能检查
```bash
# 启动开发服务器
npm run dev

# 访问页面
http://localhost:3000/order-management
http://localhost:3000/cost-config
# ... 其他页面
```

### 2. 性能检查
```bash
# 使用Chrome DevTools
1. 打开页面
2. 打开DevTools (F12)
3. 进入Performance标签
4. 点击Record按钮
5. 执行页面操作
6. 停止Record
7. 查看性能指标
```

### 3. 响应式检查
```bash
# 使用Chrome DevTools
1. 打开页面
2. 打开DevTools (F12)
3. 点击设备工具栏按钮
4. 选择不同的设备尺寸
5. 检查页面显示是否正常
```
